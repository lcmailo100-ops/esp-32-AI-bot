# esp-32-AI-bot
ESP32-based AI bot built with a microphone, audio amplifier, and speaker for voice interaction. It processes commands and responds with sound output, powered by a rechargeable battery system. Compact, easy to use, and ideal for learning IoT, embedded systems, and basic AI interaction.
